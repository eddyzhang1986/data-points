# Data-Points 前端项目

Project is loading...

## 启动

clone 仓库

```bash
$ git clone https://github.com/jiangseventeen/data-points.git
$ cd frontend
```

使用`npm`安装依赖包
```bash
$ npm install
```
or

使用cnpm安装，npm和cnpm两种方式二选一
```bash
$ npm install -g cnpm --registry=https://registry.npm.taobao.org
$ cnpm install
```


启动项目
```bash
$ npm run serve
```
浏览器打开 [http://localhost:8080](http://localhost:8080)
